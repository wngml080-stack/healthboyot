## 1. 시스템 소개

HEALTHBOYGYM OT 시스템은 헬스장의 **OT(오리엔테이션 트레이닝) 배정, 세션 관리, 매출 추적**을 위한 웹 기반 관리 도구입니다.

### 주요 기능
- 신규 회원 등록 및 OT 배정
- 트레이너별 폴더로 회원 관리
- OT 1차~3차 세션 스케줄 및 진행 관리
- 매출 대상자 추적 및 PT 전환 관리
- 실시간 알림 (신규 배정, D+3 미진행 경고 등)
- 통계 대시보드 및 월별 리포트 엑셀 다운로드
- 구글 시트 자동 연동 (FC,PT일지 ↔ Supabase)

## 2. 로그인 및 회원가입

### 로그인
1. 시스템 접속 시 **로그인 화면**이 표시됩니다.
2. 이메일과 비밀번호(6자 이상)를 입력합니다.
3. **[로그인]** 버튼을 클릭합니다.

### 신규 회원가입
1. 로그인 화면 하단의 **[회원가입하기]** 버튼을 클릭합니다.
2. **이름**, **이메일**, **비밀번호**를 입력합니다.
3. **[회원가입]** 버튼을 클릭합니다.
4. "회원가입이 완료되었습니다" 메시지가 나타납니다.
5. **관리자 승인 후** 로그인이 가능합니다.

> 승인 전에 로그인을 시도하면 "관리자 승인 대기 중입니다" 메시지가 표시됩니다.

## 3. 화면 구성 안내

### 상단 네비게이션 바 (검은색 바)
상단에 노란색 라인이 있는 검은색 바로, 다음 메뉴가 표시됩니다:

| 메뉴 | 설명 |
|------|------|
| **트레이너 관리** | 트레이너별 폴더 → 회원 카드, 스케줄, 통계 |
| **회원관리** | 전체 회원 목록 및 신규 OT 대기 현황 |
| **직원 관리** | 직원 계정 생성/수정/삭제 및 역할 부여 |
| **통계** | 매출 요약, OT 현황, 트레이너별 성과 등 |

- 우측에 **현재 로그인한 사용자 이름**과 **로그아웃 버튼**이 표시됩니다.
- 모바일에서는 햄버거 메뉴(☰)를 터치하면 메뉴가 펼쳐집니다.

- FC는 **트레이너 관리 + 회원관리**를 사용합니다.

## 5. 관리자 사용 가이드

관리자(admin/관리자)는 시스템의 **모든 기능**에 접근할 수 있습니다.

### 5-1. 직원 관리

**직원 등록하기**
1. 상단 메뉴에서 **[직원 관리]** 클릭
2. 우측 상단 **[+ 직원 추가]** 버튼 클릭
3. 이메일, 비밀번호, 이름을 입력하고 **역할**을 선택합니다.
   - 역할: 관리자 / 팀장 / 트레이너 / 강사 / FC
4. **[등록]** 클릭

**직원 정보 수정**
- 직원 목록에서 연필 아이콘(수정)을 클릭하여 이름, 역할을 변경할 수 있습니다.

**직원 승인**
- 회원가입 후 승인 대기 중인 직원은 목록에 "미승인" 상태로 표시됩니다.
- 체크 아이콘을 클릭하여 승인합니다.

**직원 삭제**
- 휴지통 아이콘을 클릭하면 삭제할 수 있습니다.

### 5-2. 트레이너 폴더 관리

**폴더 추가**
1. **[트레이너 관리]** 메뉴 클릭
2. 점선 테두리의 **[+ 폴더 추가]** 카드 클릭
3. 드롭다운에서 **트레이너를 선택**합니다.
   - 직원 관리에 등록된 승인 완료 직원만 선택 가능
4. 폴더 비밀번호를 설정합니다 (선택사항).
5. **[추가]** 클릭

**폴더 순서 변경**
- 폴더 카드 우측 상단의 **⋮ 메뉴** → "왼쪽으로 이동" 또는 "오른쪽으로 이동"

**폴더 삭제**
- 폴더 카드 우측 상단의 **⋮ 메뉴** → "폴더 삭제"
- 배정된 회원 데이터는 유지되며, 폴더만 목록에서 제거됩니다.

> 관리자는 비밀번호 없이 모든 폴더에 바로 접근할 수 있습니다.

### 5-3. OT 배정 (빠른 배정)

1. **[회원관리]** 메뉴 클릭
2. "신규 대기" 목록에서 배정할 회원을 클릭합니다.
3. **배정 팝업**에서:
   - **상태** 변경 (신청대기 → 배정완료)
   - **PT 담당** 트레이너 선택
   - **PPT 담당** 트레이너 선택 (선택사항)
4. **[저장]** 클릭

### 5-4. 통계 확인

1. **[통계]** 메뉴 클릭
2. 확인할 수 있는 항목:
   - **매출 요약**: 목표 매출 대비 달성율 게이지
   - **OT 현황**: 상태별(진행중/거부/등록완료/연락두절 등) 인원 수
   - **트레이너별 성과**: 각 트레이너의 배정/등록/매출 현황
   - **주차별 실적**: 1~5주차별 매출 추이
3. **엑셀 다운로드** 버튼으로 월별 리포트를 내려받을 수 있습니다.

### 5-5. 목표 매출 설정

- 통계 페이지에서 **목표 매출 금액**을 설정합니다.
- 설정된 목표 대비 실제 매출의 **달성율**이 게이지로 표시됩니다.

## 6. 팀장 사용 가이드

팀장은 **트레이너 관리** 페이지를 중심으로 사용합니다.

### 6-1. 트레이너 폴더 접근

1. **[트레이너 관리]** 메뉴 클릭
2. 각 트레이너별 폴더 카드가 표시됩니다.
   - 카드에는 **금일 OT**, **매출대상자**, **PT전환**, **전체회원** 수가 요약됩니다.
3. 열고 싶은 폴더를 클릭합니다.
   - 비밀번호가 설정된 폴더는 비밀번호 입력 팝업이 나타납니다.

### 6-2. 트레이너 폴더 내부 — 3개 탭

폴더를 열면 좌측에 서브메뉴가 나타나며, 3개의 탭을 전환할 수 있습니다:

#### 회원관리 탭
- 해당 트레이너에게 배정된 **모든 회원 카드**가 표시됩니다.
- 각 카드를 펼치면 **OT 세션 진행 현황**을 확인할 수 있습니다.
- 회원 카드 클릭 시 **상세 정보 페이지**로 이동합니다.

#### 스케줄 탭
- **주간 캘린더** 형태로 스케줄을 확인합니다.
- 7일간의 시간대별 OT 예약 현황을 볼 수 있습니다.

#### 통계표 탭
- 해당 트레이너의 **주간 성과 요약**이 표시됩니다.
- 매출 요약 카드: 예상 매출, 실제 매출
- OT 상태별 인원 현황

## 7. 트레이너/강사 사용 가이드

트레이너와 강사는 **자신의 폴더**에 접근하여 배정된 회원을 관리합니다.

### 7-1. 내 폴더 접근

1. **[트레이너 관리]** 메뉴 클릭
2. 자신의 이름이 적힌 폴더를 클릭합니다.
3. 비밀번호가 설정되어 있으면 비밀번호를 입력합니다.

### 7-2. 배정된 회원 확인

- **회원관리 탭**에서 나에게 배정된 회원 목록이 카드 형태로 표시됩니다.
- 각 카드에서 확인할 수 있는 정보:
  - 회원 이름, 연락처
  - OT 종목 (헬스/필라/헬스,필라/PT등록)
  - 현재 진행 상태 (OT1차예정, OT1차완료, OT2차예정 등)
  - 매출 상태

### 7-3. OT 세션 진행하기

OT는 총 **3회 세션**으로 구성됩니다.

**세션 스케줄 잡기**
1. 회원 카드를 펼칩니다.
2. 해당 세션의 **날짜/시간을 선택**합니다.
3. 저장하면 상태가 "OT1차예정" → "OT1차완료" 순으로 자동 전환됩니다.

**세션 완료 기록**
1. 세션을 진행한 후 카드를 펼칩니다.
2. **운동 내용**, **트레이너 팁**, **유산소 종류/시간** 등을 기록합니다.
3. **[완료]** 처리합니다.

**세션 진행 흐름**
```
OT1차예정 → OT1차완료 → OT2차예정 → OT2차완료 → OT3차예정 → OT3차완료
```

- 1차 저장 시 → 상태가 자동으로 "진행중"으로 전환
- 3차 완료 시 → 상태가 자동으로 "완료"로 전환

### 7-4. 스케줄 확인

- **스케줄 탭**에서 이번 주 일정을 한눈에 볼 수 있습니다.
- 시간대별로 어떤 회원의 OT가 예약되어 있는지 확인합니다.

### 7-5. 매출 관련 기록

- 회원 카드에서 **매출 상태**를 변경할 수 있습니다:
  - OT진행중 / 거부자 / 등록완료 / 스케줄미확정 / 연락두절 / 클로싱실패
- **매출 대상자** 표시, **PT 전환 희망** 체크가 가능합니다.
- 예상 매출, 실제 매출, 클로징 확률 등을 입력합니다.

### 7-6. 회원 상세 정보

회원 카드를 클릭하면 **상세 페이지**로 이동합니다:
- **회원 프로필**: 이름, 연락처, 등록 종목, 운동 시간, 부상 태그
- **타임라인**: 1차~3차 세션의 스케줄/완료 기록이 시간순으로 표시
- **상태 변경**: PT 전환, 완료, 거부 등의 상태를 변경할 수 있습니다.
- **메모/특이사항**: 자유롭게 기록

## 8. FC 사용 가이드

FC(프론트 카운터)는 **트레이너 관리**와 **회원관리** 페이지를 사용합니다.

### 8-1. 트레이너 관리

- 모든 트레이너 폴더에 **비밀번호 없이** 접근할 수 있습니다.
- 각 트레이너의 회원 현황을 확인하고 모니터링합니다.

### 8-2. 회원관리 (대시보드)

1. **[회원관리]** 메뉴 클릭
2. 상단에 요약 카드가 표시됩니다:
   - **전체 회원 수**, **대기 중 OT**, **금주 완료**, **오늘 스케줄**
3. **신규 대기 목록**: 새로 등록되어 트레이너 배정이 필요한 회원
4. **오늘의 OT**: 당일 예정된 세션 목록
5. 회원 목록에서 **검색**, **필터**(트레이너별, 상태별, 날짜별)를 사용할 수 있습니다.

### 8-3. 회원 등록

- 회원관리에서 **빠른 등록** 기능으로 신규 회원을 바로 등록할 수 있습니다.
- 이름, 연락처, 종목, 운동기간 등을 입력합니다.

## 9. OT 업무 흐름 (전체)

전체 업무의 흐름을 순서대로 정리합니다:

```
[1단계] 회원 등록
  구글 시트(FC,PT일지)에 회원 정보 입력
  또는 시스템에서 직접 등록
         ↓
[2단계] OT 신청
  구글 시트 K열(OT신청종목)에 "헬스", "필라", "헬스,필라" 입력
  → OT배정 시트에 자동 행 추가
  → OT배정 시트 G열(운동시간) 입력 시 Supabase로 자동 전송
         ↓
[3단계] 트레이너 배정
  관리자가 회원관리에서 신규 대기 회원 확인
  → PT 담당 / PPT 담당 트레이너 선택
  → 상태: "신청대기" → "배정완료"
         ↓
[4단계] OT 세션 진행 (3회)
  트레이너가 자신의 폴더에서 회원 확인
  → 1차 세션 스케줄 → 1차 완료 → 2차 → 3차
  → 각 세션마다 운동내용, 피드백 기록
         ↓
[5단계] 완료 및 매출 처리
  3차 완료 시 자동으로 "완료" 상태 전환
  → 매출 대상자 여부 확인
  → PT 전환 시 실제 매출 기록
  → 통계에 자동 반영
```

### OT 상태 흐름도

| 상태 | 색상 | 의미 |
|------|------|------|
| 신청대기 | 노란색 | 새로 신청되어 트레이너 미배정 |
| 배정완료 | 파란색 | 트레이너가 배정됨, 세션 미시작 |
| 진행중 | 초록색 | 1차 이상 세션이 진행됨 |
| 완료 | 회색 | 3차 세션까지 모두 완료 |
| 거부 | 빨간색 | 회원이 OT를 거부함 |
| 추후결정 | 보라색 | 당장은 보류, 나중에 결정 |

### 매출 상태 종류

| 매출 상태 | 의미 |
|-----------|------|
| OT진행중 | OT 세션이 아직 진행 중 |
| 거부자 | OT 또는 PT를 거부한 회원 |
| 등록완료 | PT 등록이 완료됨 |
| 스케줄미확정 | 일정이 아직 잡히지 않음 |
| 연락두절 | 회원과 연락이 안 됨 |
| 클로싱실패 | PT 전환 시도했으나 실패 |

## 10. 구글 시트 연동 안내

시스템은 **구글 시트**와 자동으로 연동됩니다.

### 연동 흐름

```
[FC,PT일지 시트]               [OT배정 시트]              [시스템(Supabase)]

E열: 등록날짜                   A열: 등록날짜
F열: 이름           K열 입력    B열: 종목
G열: 번호          ────────→   C열: 이름
H열: 종목           자동복사    D열: 연락처
J열: 운동기간                   E열: OT신청종목
K열: OT신청종목                 F열: 운동기간
                               G열: 운동시간(수동) ──→  Supabase 전송
                               H열: 운동목적(수동)
                               I열: 특이사항(수동)
```

### 구글 시트 사용법

**OT 신청 (FC,PT일지 시트)**
1. K열(OT신청종목)에 값을 입력합니다.
   - 유효 값: `헬스`, `필라`, `헬스,필라`, `필라,헬스`
2. 입력 즉시 **OT배정 시트**에 자동으로 행이 추가됩니다.
3. K열 셀이 **초록색**으로 변하면 성공입니다.

**OT 정보 전송 (OT배정 시트)**
1. 자동 생성된 행에서 **G열(운동시간)** 을 수동 입력합니다.
2. 운동시간이 입력되면 자동으로 **Supabase(시스템)로 데이터가 전송**됩니다.
3. C열(이름)이 **초록색**으로 변하면 전송 성공입니다.

**OT 취소 (삭제)**
- FC,PT일지의 K열 값을 지우면 OT배정 시트에서도 해당 행이 **자동 삭제**됩니다.

## 11. 자주 묻는 질문 (FAQ)

### Q: 로그인이 안 됩니다.
**A:** 회원가입 후 관리자 승인이 필요합니다. "관리자 승인 대기 중입니다" 메시지가 나타나면 관리자에게 승인을 요청하세요.

### Q: 메뉴를 클릭하면 "접근 권한 없음"이 나옵니다.
**A:** 역할에 따라 접근 가능한 메뉴가 다릅니다. 4번 항목의 권한표를 확인하세요. 추가 권한이 필요하면 관리자에게 역할 변경을 요청하세요.

### Q: 트레이너 폴더에 비밀번호를 잊었습니다.
**A:** 관리자에게 요청하세요. 관리자는 비밀번호 없이 모든 폴더에 접근할 수 있으며, 비밀번호를 재설정해줄 수 있습니다.

### Q: 구글 시트에 OT 종목을 입력했는데 OT배정 시트에 안 나옵니다.
**A:** 다음을 확인하세요:
- K열에 입력한 값이 정확한지 확인 (`헬스`, `필라`, `헬스,필라` 만 유효)
- 같은 행에 **이름(F열)**과 **번호(G열)**가 있는지 확인
- 셀 색상이 초록색으로 변하지 않으면 값이 유효하지 않은 것입니다.

### Q: OT 3차까지 완료했는데 상태가 안 바뀝니다.
**A:** 3차 세션을 "완료" 처리해야 자동으로 상태가 전환됩니다. 카드를 펼쳐서 3차 세션의 완료 버튼을 확인하세요.

### Q: 알림은 어디서 확인하나요?
**A:** 시스템 내 인앱 알림으로 표시됩니다. 다음 알림이 지원됩니다:
- **D+3 미진행 경고**: 배정 후 3일 이상 세션이 시작되지 않은 경우
- **신규 배정 알림**: 새로운 회원이 배정된 경우
- **당일 스케줄 알림**: 오늘 예정된 세션이 있는 경우

### Q: 엑셀 리포트는 어떻게 다운로드하나요?
**A:** 통계 페이지에서 **엑셀 다운로드** 버튼을 클릭하면 5개 시트(매출요약, 주차별, OT현황, 트레이너별, 경로별)가 포함된 파일이 생성됩니다.

### Q: 모바일에서도 사용할 수 있나요?
**A:** 네, 모바일 반응형으로 설계되어 있습니다. 스마트폰 브라우저에서 접속하면 모바일에 최적화된 화면이 표시됩니다. 상단 햄버거 메뉴(☰)로 메뉴를 열 수 있습니다.
